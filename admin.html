Cómo funciona
1️⃣  Entrá a mundial.html (botón arriba o desde el dashboard)
2️⃣  El botón ⚙️ (abajo a la derecha) abre el panel de admin
3️⃣  Elegís un partido → Abrir votación → el grupo vota el marcador
4️⃣  Cerrar votación → calcula el pronóstico grupal (más votado)
5️⃣  Cuando se juega el partido real → cargás el resultado oficial
Los grupos, la tabla de posiciones y las llaves se actualizan solos para todos en tiempo real.
Acceso rápido
